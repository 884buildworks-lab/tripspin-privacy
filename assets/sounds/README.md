# サウンドファイル配置場所

このディレクトリに以下の音声ファイルを配置してください:

## 必要なファイル

1. **spin.mp3**
   - ルーレット回転中の音
   - 推奨: ドラムロール、回転音、ワクワクする音など
   - 長さ: 4秒程度（ループ可能だとベター）

2. **complete.mp3**
   - ルーレット完了時の音
   - 推奨: チャイム、ピロリン音、ファンファーレなど
   - 長さ: 1-2秒程度

## フリー素材サイト

- 効果音ラボ: https://soundeffect-lab.info/
- DOVA-SYNDROME: https://dova-s.jp/
- 魔王魂: https://maou.audio/

## 音声ファイル配置後の設定

音声ファイルを配置したら、`utils/soundManager.ts` のコメントアウトされているコードを有効化してください。

## 現在の状態

音声ファイルがなくても、アプリは正常に動作します。
音声ON/OFF設定は機能し、ファイルが配置されると自動的に再生されます。
