## 📐 デザイン詳細

### コンセプト
**ルーレット × 日本旅行**を表現した、明るく楽しいアイコン

### デザイン要素
1. **カラフルなルーレット盤**: 8色のセグメントで回転感を表現
2. **日本列島シルエット**: 中央に配置、旅行アプリであることを明示
3. **矢印ポインター**: 上部にルーレットの針を配置
4. **キラキラエフェクト**: 4つの星で楽しさを演出
5. **グラデーション背景**: 赤〜オレンジの明るいグラデーション

### カラーパレット
- **背景**: #FF8E53 → #FF6B6B (グラデーション)
- **ルーレット8色**:
  - #FF6B6B (赤)
  - #4ECDC4 (青緑)
  - #FFD93D (黄色)
  - #A29BFE (紫)

## 🔄 アイコンの再生成方法

アイコンのデザインを変更したい場合：

### 1. SVGを編集
`assets/images/icon-source.svg` を編集してください。

- デザインツール: Figma, Illustrator, Inkscapeなど
- または、SVGコードを直接編集

### 2. アイコン再生成
```bash
npm run generate-icons
```

これで全サイズのPNGが自動生成されます。

## 📦 Google Play / App Store 用

### Google Play Console
1. **アプリアイコン**: `assets/icon-512.png` をアップロード
2. **機能グラフィック**: 別途作成が必要 (1024x500px)
3. **スクリーンショット**: アプリの画面キャプチャをアップロード

### App Store Connect
1. **アプリアイコン**: `assets/icon.png` (1024x1024) をアップロード
2. **スクリーンショット**: 各デバイスサイズごとに準備

## 🎯 アイコンのベストプラクティス

### ✅ Good
- シンプルで認識しやすいデザイン
- 小さいサイズでも視認可能
- アプリの目的が一目でわかる
- カラフルで目立つ配色

### ❌ Avoid
- 文字を入れない（小さいサイズで読めない）
- 細かすぎるディテール
- 暗すぎる色使い
- 他のアプリと似たデザイン

## 🛠 トラブルシューティング

### アイコンが表示されない
```bash
# キャッシュをクリア
npx expo start -c
```

### Androidでアイコンが切れる
Android Adaptive Iconは外周20%がマスクされます。
重要な要素は中央80%に配置してください。

### アイコン生成が失敗する
sharpパッケージを再インストール：
```bash
npm install --save-dev sharp
```
